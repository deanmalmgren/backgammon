# backgammon

Simulate backgammon games

## Getting started

1. Make sure the current working directory is on the python path.
   ```bash
   echo PYTHONPATH=:`pwd` > .env
   ```

1. Install dependencies
   ```bash
   poetry install
   ```

1. Run tests
   ```bash
   poetry run nose2
   ```

1. Run example
   ```bash
   poetry run python example.py
   ```
