# backgammon

Simulate backgammon games.

## Getting started

1. Configure the development environment
   ```bash
   echo PYTHONPATH=:`pwd` > .env
   ```

1. Install dependencies
   ```bash
   poetry install
   ```

1. Run tests
   ```bash
   poetry run nose2
   ```

1. Run example
   ```bash
   poetry run python example.py
   ```
